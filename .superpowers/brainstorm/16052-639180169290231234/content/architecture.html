核心数据流

打卡流程

  1. 家长在后台设置学习任务
  2. 孩子在小程序查看今日任务
  3. 孩子拍照/上传学习成果
  4. 家长审核确认（可选）
  5. 打卡成功，连击+1，获得积分

奖励流程

  1. 连续打卡3天 → 解锁小奖励
  2. 连续打卡7天 → 解锁中奖励
  3. 完成任务树节点 → 领取成就奖励
  4. 积分累积 → 兑换实物奖励
  5. 家长线下发放奖励

未打卡惩罚机制

⚠️ 未打卡处理：

  • 当日 19:00 自动推送"催打卡提醒"
  • 连续漏打 2 天：连击重置为 0
  • 连续漏打 3 天：扣除已获积分的 10%
  • 特殊情况下家长可手动"补卡"
