## 1. 项目范围

### 1.1 交付物

| 层级   | 模块      | 说明              |
| ---- | ------- | --------------- |
| 小程序端 | 首页-今日任务 | 任务列表、打卡入口、连击展示  |
| 小程序端 | 任务树     | 成就解锁树形结构、进度展示   |
| 小程序端 | 奖励中心    | 积分余额、奖励兑换、兑换记录  |
| 小程序端 | 我的      | 个人中心、打卡记录、成就展示  |
| 后台管理 | 登录/退出   | 账号密码登录、JWT鉴权    |
| 后台管理 | 首页概览    | 数据统计卡片、今日打卡列表   |
| 后台管理 | 孩子管理    | 添加/编辑/删除孩子、头像上传 |
| 后台管理 | 任务设置    | 按孩子配置任务、周期、积分   |
| 后台管理 | 奖励管理    | 奖励分类、增删改查       |

- 多用户/多角色权限管理（仅家长单角色）
- 班级/教师/机构功能
- 在线支付购买积分
- 消息推送（微信模板消息需上线后配置，留接口）
- 数据分析图表（仅展示基础统计数据）

## 3. 里程碑与开发顺序

### 里程碑一：项目初始化与基础架构

**目标：** 前后端项目骨架搭建完成，可运行

**任务清单：**

1. 后端 Spring Boot 项目初始化
   
   - 创建 Maven 项目结构
   - 配置 pom.xml 依赖（Spring Boot、MyBatis-Plus、Redis、JWT、Lombok）
   - 配置 application.yml（MySQL、Redis、端口）
   - 创建启动类、配置类、统一响应类

2. 前端 admin-web 项目初始化
   
   - 使用 Vite 创建 Vue 3 项目
   - 安装 Element UI、Vue Router、Pinia、Axios
   - 配置路由、状态管理、Axios 封装
   - 创建布局组件（侧边栏 + 头部）

3. 数据库初始化
   
   - 创建数据库表结构 SQL 脚本
   - 初始化管理员账号

## 6. 风险与注意事项

| 风险项     | 影响      | 应对措施                  |
| ------- | ------- | --------------------- |
| 照片存储    | 文件存储成本高 | 接入 OSS，本地开发用本地存储模拟    |
| 微信小程序上线 | 需要企业资质  | V1 先做功能开发，上线流程后续处理    |
| 并发打卡    | 重复提交    | Redis 分布式锁 + 幂等校验     |
| 连击计算准确性 | 数据不一致   | 每天凌晨定时任务校准，Redis+DB双写 |
| 积分安全性   | 积分被刷    | 所有积分变更走服务层统一方法，记录流水   |

## 7. 开发顺序建议

按以下顺序逐步交付，每完成一个里程碑可独立验证：

1. **里程碑一**：项目初始化（可运行的空项目）
2. **里程碑二**：认证 + 孩子管理（第一个可交互模块）
3. **里程碑三**：任务 + 打卡 + 审核（核心业务闭环）
4. **里程碑四**：奖励 + 兑换 + 成就（激励体系）
5. **里程碑五**：小程序端（孩子端全部功能）
6. **里程碑六**：联调优化
